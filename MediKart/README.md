# MediKart

MediKart is a full-stack e-commerce web application for medicines and healthcare products. The platform allows users to browse healthcare products, search items, manage carts, place orders, and interact with an admin dashboard for product and category management.

## Features

* Product listing and category management
* Search functionality
* Shopping cart system
* User authentication
* Order management
* Admin dashboard
* Prescription upload support
* Responsive user interface

## Tech Stack

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Spring Boot
* Java

### Database

* SQL
